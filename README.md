# Blog frontend

WIP blog frontend for a family member.

## Stack

#### Frontend
- React
- Frameworks: React Router (Vite) with TailwindCSS and ShadCN
- Static server ran with Caddy (no React SSR)

#### Backend
- Linode / AKAMAI PROTECTION
- Caddy
- CaddyGuard / PROTECTION
- Go (web server / stream)
- MongoDB
